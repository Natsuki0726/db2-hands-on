# Db2 SQL ハンズオン

## 1. IBM Cloud にアクセス

1. 以下のURLにアクセスします。  
   https://cloud.ibm.com

2. IBM Cloud にログインします。

3. ログイン後、**`db2-mc_HandsON`** という環境に入ってください。

---

# 2. テーブルが開けない場合（整合性チェック）

データをインポートした後、テーブルを開けない場合があります。  
その場合は、以下のSQLを実行して **テーブルの整合性チェック** を行ってください。

```sql
SET INTEGRITY FOR 商品 IMMEDIATE CHECKED;
SET INTEGRITY FOR 顧客 IMMEDIATE CHECKED;
SET INTEGRITY FOR 購入 IMMEDIATE CHECKED;
```

---

# 3. データの取得・集計

## 3.1 データを取得する（SELECT）

テーブルからデータを取得します。

```sql
SELECT 商品名, 価格 FROM 商品;

SELECT * FROM 商品;

SELECT 商品名 AS Name, 価格 AS price FROM 商品;
```

---

## 3.2 条件を指定してデータを取得する（WHERE）

条件を指定してデータを取得します。

```sql
SELECT 商品名, 価格 
FROM 商品 
WHERE 価格 >= 500 
ORDER BY 価格;
```

```sql
SELECT 商品名, 価格 
FROM 商品 
WHERE カテゴリ <> '食料品' 
ORDER BY 価格;
```

---

## 3.3 集約関数を使用する（GROUP BY）

カテゴリごとに商品数や平均価格を集計します。

```sql
SELECT 
    カテゴリ,
    COUNT(*) AS 個数,
    AVG(価格) AS 平均価格
FROM 商品
GROUP BY カテゴリ;
```

---

## 3.4 テーブルを結合する（INNER JOIN）

購入テーブルと顧客テーブルを結合し、購入IDと顧客名を取得します。

```sql
SELECT 
    購入.購入ID,
    顧客.氏名
FROM 購入
INNER JOIN 顧客 
ON 購入.USERID = 顧客.USERID;
```

# 4. データの追加・更新・削除

## 4.1 データが存在しないことを確認する

まず、`USERID = 51` のデータが存在しないことを確認します。

```sql
SELECT * 
FROM 顧客 
WHERE USERID = 0051;
```

---

## 4.2 データを挿入する（INSERT）

顧客データを追加します。

```sql
INSERT INTO 顧客 
(USERID, 氏名, メールアドレス, 電話番号)
VALUES 
(0051, '好きな名前を入れてください', '適当なメールアドレス', '電話番号');
```

確認します。

```sql
SELECT * 
FROM 顧客 
WHERE USERID = 0051;
```

---

## 4.3 データを更新する（UPDATE）

電話番号を更新します。

```sql
UPDATE 顧客
SET 電話番号 = '080-6527-9271'
WHERE USERID = 0051;
```

確認します。

```sql
SELECT * 
FROM 顧客 
WHERE USERID = 0051;
```

---

## 4.4 データを削除する（DELETE）

顧客データを削除します。

```sql
DELETE FROM 顧客
WHERE USERID = 0051;
```

確認します。

```sql
SELECT * 
FROM 顧客 
WHERE USERID = 0051;
```

# 追加練習問題（早く終わった人向け）

時間が余った方は、以下の問題に挑戦してみてください。  
まずはヒントを見ずに挑戦してみましょう。

---

## 問題1：高い商品を探す

**価格が1000円以上の商品**を取得してください。

取得する項目

- 商品名
- 価格

<details>
<summary>ヒントを見る</summary>

- `WHERE`
- `>=`

</details>

<details>
<summary>回答例を見る</summary>

```sql
SELECT 商品名, 価格
FROM 商品
WHERE 価格 >= 1000;
```

</details>

---

## 問題2：カテゴリごとの商品数

**カテゴリごとの商品数**を集計してください。

取得する項目

- カテゴリ
- 商品数

<details>
<summary>ヒントを見る</summary>

- `GROUP BY`
- `COUNT(*)`

</details>

<details>
<summary>回答例を見る</summary>

```sql
SELECT カテゴリ, COUNT(*) AS 商品数
FROM 商品
GROUP BY カテゴリ;
```

</details>

---

## 問題3：一番高い商品

**最も価格が高い商品**を取得してください。

取得する項目

- 商品名
- 価格

<details>
<summary>ヒントを見る</summary>

- `ORDER BY`
- `DESC`
- `FETCH FIRST`

</details>

<details>
<summary>回答例を見る</summary>

```sql
SELECT 商品名, 価格
FROM 商品
ORDER BY 価格 DESC

```

</details>

## 問題5（チャレンジ）

顧客ごとに **購入回数** を集計してください。

取得する項目

- 顧客名
- 購入回数

<details>
<summary>ヒントを見る</summary>

- `JOIN`
- `GROUP BY`
- `COUNT(*)`

</details>

<details>
<summary>回答例を見る</summary>

```sql

```

</details>

## ボーナス問題

**一番多く購入している顧客**を取得してください。

<details>
<summary>ヒントを見る</summary>

- `GROUP BY`
- `COUNT`
- `ORDER BY`
- `DESC`

</details>

<details>
<summary>回答例を見る</summary>

```sql
SELECT 顧客.氏名, COUNT(*) AS 購入回数
```
